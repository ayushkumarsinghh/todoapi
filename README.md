# 📝 Todo CRUD API

A simple backend API built using Node.js and Express that supports full CRUD (Create, Read, Update, Delete) operations for managing todos.

---

## 🚀 Features

* Create a new todo
* Get all todos
* Get a todo by ID
* Update a todo
* Mark a todo as done
* Delete a todo

---

## 🛠️ Tech Stack

* Node.js
* Express.js
